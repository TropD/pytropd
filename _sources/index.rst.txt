===================================
Welcome to PyTropD's documentation!
===================================

PyTropD is a python package intended to calculate different metrics of tropical width as described in `Adam et al. (2018)`__. This package is the python equivalent to the original MATLAB |registered| version

.. |registered| unicode:: U+00AE .. registered trademark sign

__ https://www.geosci-model-dev-discuss.net/gmd-2018-124/

This documentation will describe version 1.0.5.


Download and Install
----------------------
See the :doc:`install` page for all the details. The project is located in a Github repository_.

.. _repository: https://github.com/tropd/pytropd

Documentation
----------------------
There is a step by step :doc:`tutorial` intended to introduce you to many
aspects of the code.

There is also a :doc:`reference` with full details.


About PyTropD
-----------------------------------
The MATLAB |registered| version was originally developed by the `ISSI Tropical Width Diagnostics Intercomparison Project`__.

 __ http://www.issibern.ch/teams/twdip/


The code was translated to Python by Alison Ming with help from Paul William Staten and TropD team.
