Installation¶

Packages¶

You’ll need to install the following packages:

  • Python 2.7
  • Numpy and Scipy
  • Matplotlib
  • `PyTropD <https://github.com/tropd/pytropd/`_

From source¶

  1. Clone the repository using
git clone git@github.com:tropd/pytropd.git
  1. Use pip for a local install

cd pytropd

pip install .

  1. Alternatively compile and build the source

cd pytropd

python setup build

sudo python setup.py install
