Welcome to PyTropD’s documentation!¶

PyTropD is a python package intended to calculate different metrics of tropical width as described in Adam et al. (2018). This package is the python equivalent to the original MATLAB ® version

This documentation will describe version 1.0.5.

Download and Install¶

See the Installation page for all the details. The project is located in a Github repository.

Documentation¶

There is a step by step Tutorial intended to introduce you to many aspects of the code.

There is also a List of metrics with full details.

About PyTropD¶

The MATLAB ® version was originally developed by the ISSI Tropical Width Diagnostics Intercomparison Project.

The code was translated to Python by Alison Ming with help from Paul William Staten and TropD team.
